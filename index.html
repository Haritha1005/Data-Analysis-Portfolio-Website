Insurance Data Analysis using Python

This project involved analyzing an insurance dataset to explore the relationship between various features and medical insurance charges. Using Python and libraries like Pandas, Matplotlib, and Seaborn, I conducted a comprehensive analysis of the dataset. Key steps included checking for missing values, visualizing the distribution of key features like BMI and charges, and examining the impact of smoker status on premium charges. The analysis revealed a positive correlation between age and insurance charges, with a notable difference between smokers and non-smokers in terms of premium increases. Various plots such as count plots, scatter plots, box plots, and line plots were utilized to uncover insights, which will help in creating a predictive model for estimating medical insurance costs.
